CDN3: The CDN solution for Web3 era 

Problem and Solution

  • A Storage Solution is NOT a CDN Solution.
  • IPFS is not related with Web3 at all, Itâ€™s just a distributed storage solution.
  • How Hosts(CDN Servers) get paid is a problem.
  • Payment channel to the rescue!
  • Payment channel is just like Stored-value card: charge once, pay many times instantly with no transaction costs.
  • Host splits data into many small blocks, then send data and get paid block by block.
  • CDN3 is a complete CDN solution for web3 using Payment channel.

Main Features

  • Data source and be any web3 storage solution as well as traditional HTTP server.
  • Optimized for large chunk data delivery: movies, short videos, eBooks, etc.
  • Hosts get paid via payment channel on Blockchain.
  • Simple HTTP Interface for client access.
  • Signer wallet for safe payment channel operation.

Payment options

  • Free with limits (bandwidth, quality).
  • Paid by client.
  • Paid by project (dApp).

Optimized Blockchain:

  • PoS secured Blockchain with extremely low gas fee and fast transaction times.
  • Built-in payment channel support.
  • Permissionless architecture.
  • Hosts can join or leave network anytime.
  • Clients can choose their hosts freely.
